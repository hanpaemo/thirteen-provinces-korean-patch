# 十三州行动 三国搜打撤 한국어 번역 개선 패치

![삼삼주 보물 전투록](main.png)

**十三州行动 三国搜打撤** (삼삼주 행동: 삼국 수색섬멸전)의 한국어 번역 개선 패치입니다.

게임에 기본 한국어가 포함되어 있지만 번역 품질이 낮아, 중국어 원문과 일본어 참고를 대조하여 전면 재번역하였습니다.

- **게임**: [十三州行动 三国搜打撤](https://store.steampowered.com/app/3684500/) (Steam)
- **엔진**: RPG Maker MZ
- **번역 항목**: 1,938개 키 전수 재번역
- **감수**: 8차 무한 감수 (번역 오류 0건 수렴) + 말투 감수 + 한자어 가독성 감수

## 주요 개선 사항

- 중국어 원문 1:1 대조 전면 재번역
- 삼국지/무협 용어 한국 관습에 맞게 통일 (부곡→병력, 후근→보급, 종자→수행원 등)
- 화자별 말투 일관성 확보 (초월자/장수/적장/서사/시스템 각각 통일)
- 난해한 중국어 음독 한자어를 자연스러운 한국어로 대체
- 시스템/도움말 텍스트 말투 평서체 통일

## 설치 방법

1. [Releases](../../releases) 페이지에서 `main.json`을 다운로드합니다.
2. 게임 설치 폴더를 엽니다.
   - Steam → 게임 우클릭 → 관리 → 로컬 파일 탐색
3. `languages/ko/main.json` 파일을 백업합니다.
4. 다운로드한 `main.json`을 `languages/ko/main.json`에 덮어씌웁니다.
5. 게임을 실행합니다.

## 제거 방법

백업해둔 원본 `main.json`을 복원하거나, Steam에서 "게임 파일 무결성 검사"를 실행합니다.

## 주의사항

- 게임 업데이트 시 `main.json`이 원본으로 덮어씌워질 수 있습니다.
- 문제가 있으면 [Issues](../../issues)에 제보해주세요.
